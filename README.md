Classroom Session Monitoring System

Project Overview

This project is designed to monitor classroom sessions in skill training programmes. It helps manage students, trainers, batches, attendance, sessions, and reports.

Features

- Student Management
- Trainer Management
- Batch Management
- Attendance Tracking
- Session Monitoring
- Reports Dashboard

Technologies Used

- HTML
- CSS
- JavaScript
- Python (Backend - Planned)
- SQLite (Database - Planned)

Project Structure

- Frontend
- Backend
- Database
- Documentation
